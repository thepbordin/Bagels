Background thread pattern: Managers spawn a daemon thread calling _trigger_entity_export() or _trigger_export_and_commit(). The thread runs immediately. Tests must wait for it to finish.

Waiting strategy: After the CRUD call, join all non-main daemon threads:
```python
import threading, time
time.sleep(0.5)  # allow daemon thread to start and complete
# OR: join threads by name pattern if needed
```
The simpler approach: use time.sleep(0.5) after each CRUD call since daemon threads are fast (no git in this config).

Location isolation:
```python
from bagels.locations import set_custom_root

# cleanup:

```

```python

```

**Test functions:**

`test_create_account_writes_yaml(db_with_temp_root)`:
- Call `create_account({"name": "Integration Bank", "beginningBalance": 999.0, "hidden": False})`
- `time.sleep(0.5)` to let daemon thread finish
- Assert: (db_with_temp_root / "data" / "accounts.yaml").exists()
- Load yaml, assert the account slug key exists and name == "Integration Bank"

`test_update_account_writes_yaml(db_with_temp_root)`:
- create_account({"name": "To Update", "beginningBalance": 100.0, "hidden": False}) → capture account.id
- time.sleep(0.5)
- update_account(account_id, {"name": "Updated Name"})
- time.sleep(0.5)
- Load accounts.yaml, assert the account name is "Updated Name" in the YAML data

`test_delete_account_writes_yaml(db_with_temp_root)`:
- create_account({"name": "To Delete", "beginningBalance": 0.0, "hidden": False}) → capture slug
- time.sleep(0.5)
- delete_account(account_id)
- time.sleep(0.5)
- Load accounts.yaml. Assert the deleted account slug is NOT in the data (export reflects deletion).

`test_create_category_writes_yaml(db_with_temp_root)`:
- from bagels.models.category import Nature
- create_category({"name": "Integration Cat", "nature": Nature.NEED, "color": "#123456"})
- time.sleep(0.5)
- Assert: (db_with_temp_root / "data" / "categories.yaml").exists()
- Load yaml, assert category present.

`test_create_person_writes_yaml(db_with_temp_root)`:
- create_person({"name": "Integration Person"})
- time.sleep(0.5)
- Assert: (db_with_temp_root / "data" / "persons.yaml").exists()
- Load yaml, assert person present.

`test_create_template_writes_yaml(db_with_temp_root)`:
- First create an account (templates require accountId). Get account.id.
- create_record_template({"label": "Monthly Rent", "amount": 1200.0, "accountId": account.id, "isIncome": False, "ordinal": 0})
- time.sleep(0.5)
- Assert: (db_with_temp_root / "data" / "templates.yaml").exists()

`test_create_record_writes_monthly_yaml(db_with_temp_root)`:
- Create prerequisite account + category.

- Assert: (db_with_temp_root / "data" / "records" / "2026-03.yaml").exists()
- Load yaml, assert records key is present and at least one record exists.

`test_delete_record_writes_yaml(db_with_temp_root)`:
