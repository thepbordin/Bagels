Create financial summary calculation functions:

1. **Monthly summary** (implements CLI-03):
   ```python
   def calculate_monthly_summary(session, month: str | None = None) -> dict
   ```
   - Parse month using parse_month() from filters.py (default: current month)
   - Query records for the month using get_records()
   - Calculate totals:
     * total_income: Sum of records where isIncome=True
     * total_expenses: Sum of records where isIncome=False
     * net_savings: total_income - total_expenses
     * record_count: Number of records
   - Return dict with keys: month, total_income, total_expenses, net_savings, record_count

2. **Budget status** (prepares for LLM-05):
   ```python
   def calculate_budget_status(session, month: str | None = None) -> dict
   ```
   - Query categories with monthly budgets
   - Calculate spent per category from records
   - Calculate remaining: budget - spent
   - Return dict with category budgets and progress

3. **Helper function**:
   ```python
   def calculate_income_expense(records: list[Record]) -> tuple[float, float, float]
   ```
   - Separate income and expense records
   - Return (total_income, total_expenses, net_savings)

Use get_records() from managers/records.py. Use get_start_end_of_month() from managers/utils.py.
