Create LLM context dump command (implements LLM-01, LLM-04, LLM-05):

1. **Command structure**:
   ```python
   @click.group()
   def llm():
       """LLM integration commands."""
       pass

   @llm.command("context")
   @click.option("--month", "-m", help="Month for context (YYYY-MM), defaults to current")
   @click.option("--period", help="Time period: all, 30d, 60d, 90d")
   @click.option("--days", type=int, help="Number of recent days to include")
   def llm_context(month, period, days):
       """Dump financial snapshot for LLM consumption."""
   ```

2. **Context structure** (implements LLM-04, LLM-05):
   ```python
   context = {
       "snapshot_date": datetime.now().isoformat(),
       "period": month or period or "current_month",
       "accounts": [...],  # List of all accounts with balances
       "summary": {...},   # Monthly summary (income, expenses, savings)
       "spending_by_category": [...],  # Category breakdown
       "recent_records": [...],  # Records for the period (last 30 or specified month)
       "budget_status": [...],  # Budget progress per category
       "categories": [...],  # Full category tree
   }
   ```

3. **Implementation**:
   - Determine time range (month default, or period/days override)
   - Query accounts: get_accounts(session)
   - Calculate summary: calculate_monthly_summary(session, month)
   - Calculate spending: calculate_spending_by_category(session, month)
   - Query recent records: get_records() for the period
   - Calculate budget status: calculate_budget_status(session, month)
   - Query categories: get_category_tree(session)

4. **Output format** (implements LLM-01):
   - Use YAML format (structured, LLM-parseable)
   - Use to_yaml() from formatters.py
   - Include descriptive comments in YAML (e.g., "# Financial snapshot for LLM analysis")
   - Ensure all nested objects are serialized (category.name, account.name)

5. **Error handling**:
   - Validate only one of month/period/days is specified
   - Handle missing data gracefully (empty sections)

Reuse functions from summaries.py, spending.py, formatters.py.
