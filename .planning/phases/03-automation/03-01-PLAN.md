    Step 1 — Add GitConfig to config.py:
    - Add `class GitConfig(BaseModel)` before `class Config` with fields:
      enabled: bool = False
      auto_commit: bool = True
      auto_push: bool = False
      auto_pull: bool = False
      remote: str = "origin"

    - Module-level `_GIT_LOCK = threading.Lock()` for serializing all index-write operations
    - `_get_repo() -> Repo | None` — opens repo at data_directory(), returns None on InvalidGitRepositoryError
    - `auto_commit_yaml(filepath: Path, message: str) -> bool` — acquires lock, index.add relative path, checks is_dirty(index=True), calls index.commit(message), returns True/False (never raises)
    - `push_to_remote(remote_name: str = "origin", branch: str = "main") -> bool` — acquires lock, remote.push(branch), returns True/False
    - `pull_from_remote(remote_name: str = "origin", branch: str = "main", silent: bool = False) -> bool` — acquires lock, remote.pull(branch), returns True/False (respects silent flag)
    - `get_status() -> list[str]` — no lock needed (read-only), returns changed + untracked file paths
    - `get_log(max_count: int = 10) -> list[dict]` — no lock needed, returns list of {hash, message, date, author} dicts
